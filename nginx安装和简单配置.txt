1. nginx安装和简单配置
	#首先从nginx官网下载源码包
	tar -tf xx.tar.gz #不解压，查看tar.gz文件列表
	tar -C ./ -xzvf nginx-1.15.7.tar.gz
	./configure #会报错，需要安装相关依赖
	
	yum install -y zlib zlib-devel
	#查看程序安装情况
	rpm -qa | grep zlib

	./configure 
	make
	make install
	
	./nginx
	./nginx stop
	./nginx relod
	
	#查看80端口被谁占用
	#netstat依次显示 
	#Proto Recv-Q Send-Q Local Address               Foreign Address             State       PID/Program name
	netstat -anp | grep 80
	netstat -ntpl | grep 80   #tcp端口
	lsof -i:80
	fuser -v -n tcp 80
	killall httpd /service httpd stop
	
	
	nginx配置两个虚拟主机都使用80端口（通过不同域名来区分）
	vim conf/nginx.conf
	配置如下：
    server {
        listen       80;
        server_name  test1.bkunzhang.com;


        location / {
            root   html1;
            index  index.html index.htm;
        }

        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }

    }

    server {
        listen       80;
        server_name  test2.bkunzhang.com;


        location / {
            root   html2;
            index  index.html index.htm;
        }

        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }

    }
	
	修改本地hosts:
	192.168.230.132 test1.bkunzhang.com
	192.168.230.132 test2.bkunzhang.com
	
	本地访问test1.bkunzhang.com，test2.bkunzhang.com成功！
